== Lan-CDS ==

This application is designed as a Content Distribution System for Local Area Networks. It connects to the uTorrent web API of a single host and pulls the list of available torrents. These are displayed in a list and provide an easy to use interface for downloading selected applications.

Required libraries:
*MonoTorrent
*MonoTorrent.Dht
*Cleverscape.UTorrentClient (included)
